# Web Frontend

Next.js 16 app with the real-time chat UI.

See the [root README](../../README.md) for setup, testing, and project overview.

## Local development

```bash
pnpm dev
```

## Tests

```bash
pnpm test          # unit tests (Vitest)
pnpm test:pages    # page-level tests
pnpm typecheck     # TypeScript check
pnpm lint          # ESLint
pnpm build         # production build
```
