Junior Backend / Full-stack Developer

I build production web applications with TypeScript, NestJS, Next.js, React, and PostgreSQL. My main portfolio project is LetsChat — a full-stack, real-time team collaboration app deployed to Vercel and Render.

## 4. Featured project description for LetsChat

**Title:** LetsChat — production real-time team collaboration app

**Description:**

Full-stack Slack-like team chat rebuilt from the archived lets-chat open-source project.

- Backend: NestJS + Prisma + PostgreSQL
- Frontend: Next.js 16 + React 19 + Tailwind CSS
- Real-time: Socket.io with membership revalidation
- Auth: JWT rotation, bcrypt, session management
- Features: workspaces, channels, DMs, group chats, invite links, contacts, file attachments, global search, push notifications, PWA
- Quality: 1,600+ tests, CI/CD, production verifiers
