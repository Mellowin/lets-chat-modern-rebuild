## 4. Project description for recruiters

Use this when a recruiter asks “What is LetsChat?”:

> LetsChat is a production full-stack team chat and collaboration app I rebuilt from an archived open-source project. It has workspaces, public/private channels, direct messages, group chats with expiring invite links, a private contacts list, authenticated file attachments, global search, Web Push notifications, and PWA installability. The backend is NestJS + Prisma + PostgreSQL, the frontend is Next.js 16 + React 19 + Tailwind CSS, and real-time messaging runs over Socket.io. The project is covered by 1,600+ automated tests and deploys automatically through GitHub Actions, Render, and Vercel.
