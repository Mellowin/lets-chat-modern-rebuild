Add this to your `Mellowin/Mellowin` (or similar) profile README:

```markdown
## 🚀 Featured Project: Lets Chat

A full-stack, real-time team collaboration app I built as my main portfolio project.

- **Live demo:** https://lets-chat-web.vercel.app
- **Repo:** https://github.com/Mellowin/lets-chat-modern-rebuild
- **Stack:** NestJS · PostgreSQL · Prisma · Next.js 16 · React 19 · Tailwind CSS · Socket.io
- **Highlights:**
  - Workspace/channel/DM collaboration with OWNER/ADMIN/MEMBER roles.
  - JWT auth with token rotation and per-tab session management.
  - Secure file attachments with authenticated downloads and Cyrillic filename support.
  - 1,500+ tests, CI/CD with GitHub Actions, and API E2E tests on PostgreSQL.

[See screenshots and demo flow →](https://github.com/Mellowin/lets-chat-modern-rebuild#-screenshots)
```
