Junior Backend / Full-stack Developer

I build web applications with TypeScript, NestJS, Next.js, React, and PostgreSQL. My main portfolio project is a full-stack, real-time team chat app inspired by Slack — workspaces, public/private channels, direct messages, authenticated file attachments, global search, and multi-device session management.

I am looking for Junior Backend or Full-stack Junior roles where I can keep growing while contributing to real products.

Live demo: https://lets-chat-web.vercel.app
GitHub: https://github.com/Mellowin/lets-chat-modern-rebuild
